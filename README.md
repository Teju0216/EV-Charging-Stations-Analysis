# ⚡ EV Charging Station Analysis Dashboard

## 📌 Project Overview

The **EV Charging Station Analysis Dashboard** is an interactive Power BI project created to analyze Electric Vehicle (EV) charging station data and generate meaningful business insights.

This dashboard helps understand:

* EV charging station distribution
* Charger type usage
* Top charging operators
* Renewable energy adoption
* EV infrastructure growth over time
* User engagement and charging trends

The project was built using **Microsoft Power BI** with DAX calculations, KPI cards, interactive charts, slicers, and map visualizations.

# 📂 Dataset Information

The dataset contains EV charging station information such as:

* Station ID
* Charger Type
* Station Operator
* Installation Year
* Charging Cost
* User Statistics
* Ratings
* Renewable Energy Source
* Latitude & Longitude

Dataset Source: Kaggle

# 📊 Dashboard Features

## KPI Cards

* Total Stations
* Average Users
* Average Cost
* Average Rating
* Renewable Energy Percentage

---

## Visualizations Used

### 🔹 Donut Chart

Analyzes charger type distribution.

### 🔹 Bar Chart

Shows top EV charging station operators.

### 🔹 Line Chart

Displays EV station growth over time.

### 🔹 Pie Chart

Analyzes renewable energy usage.

### 🔹 Map Visualization

Shows charging station locations using latitude and longitude.

### 🔹 Interactive Slicers

Used for filtering dashboard data dynamically.

---

# 🎯 Key Insights

* AC Level 2 chargers are the most commonly used.
* EV charging infrastructure has grown steadily over the years.
* Renewable energy adoption is increasing among charging stations.
* Certain operators dominate the EV charging market.
* User engagement varies significantly by charger type and location.
